Boolean values

Boolean only take the values true or false

Boolean() function

Boolean(10 > 9); // returns true
(10 > 9) ; // also returns true
10 > 9 ; // also returns true

Everything with a "Value" is True

Boolean(100); // returns true

Boolean(3.14); // returns true

Boolean(-15); // returns true

Boolean("Hello"); // returns true

Boolean("false"); // returns true

Boolean(7 + 1 + 3.14); // returns true

Everything without a "Value" is False

Boolean(0); // 0 returns false

Boolean(-0); // returns false

Boolean(""); // empty string returns false

var x;
Boolean(x); // undefined returns false

Boolean(false); // false returns false

Boolean(NaN); // NaN returns false

Booleans can be Objects

Do not create Boolean objects. It slow down execution speed

var x = false; // typeof x returns boolean
var y = new Boolean(false); // typeof y returns object
