Note: Some methods return a list of HTML Elements, which can be accessed like array

Exp: elements[0]

But, it is not an array, which mean you can not use array funtion on it (pop(), push(), join(),...)


getElementById()

This example finds the element with id="intro":

var myElement = document.getElementById("intro");

getElementByTagName()

This example finds all <p> elements:

var myElements = document.getElementByTagName("p");

This example finds the element with id="main", and then finds all <p> elements inside "main"

var x = document.getElementById("main");
var myElements = x.getElementByTagName("p");

getElementByClassName()

var myElements = document.getElementByClassName("intro");

querySelectorAll()

Find all <p> elements

var x1 = document.querySelectorAll(".example");

Find all class="example" elements

var x1 = document.querySelectorAll("p");

Find all <p> with class="example" elements

var x1 = document.querySelectorAll("p.example");

Find all <a> elements that have a "target" attribute

var x1 = document.querySelectorAll("a[target]");

Find all <p> elements where the parent is a <div> elements

var x1 = document.querySelectorAll("div > p");

Find all <h2>, <div>, <p> elements

var x1 = document.querySelectorAll("h2, div, p");

Note: Use x1[0] to get / set the first element in x1
