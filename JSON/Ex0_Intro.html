JSON: JavaScript Object Notation

JSON is a syntax for stroing and exchanging data

JSON is text, written with JavaScript object notation

Exchanging Data

When exchangign data between a browser and a server, the data can only be text.

JSON is text, & we can convert any JavaScript object into JSON, and send JSON to the server.

We can also convert JSON received from theserver into JavaScript object.

This way we can work with the data as JavaScript objects, with no complicated parsing and translations

Sending Data

If you have data stord in a JavaScript object, you can oconvert hte object into JSON, and send it to a server:

var myObj = {name: "John", age: 31, city: "New York"};
var myJSON = JSON.stringify(myObj);
window.location = "demo_json.php?x=" + myJSON;

Receiving Data

If you receive data in JSON format, you can convert it into a JavaScript object:

var myJSON = {name: "John", age: 31, city: "New York"};
var myObj = JSON.parse(myJSON);
document.getElementById("demo").innerHTML = myObject.name;

Storing data

When storing data, the data has to be a certain format, and regardless of where you choose to store it, text is always one of the legal formats.

JSON makes it possible to store JavaScript objects as text.

// Storing data:
var myObj = {name: "John", age: 31, city: "New York"};
var myJSON = JSON.stringify(myObj);
locaStorage.setItem("textJSON", myJSON);

// Storing data:
text = localStorage.getItem("textJSON");
obj = JSON.parse(text);
document.getElementById("green").innerHTML = obj.name;

What is JSON?

  • JSON: JavaScript Object Notation
  • JSON is a lightweight data-interchange format
  • JSON is "self-describing" and easy to understand
  • JSON is language independent *

*

JSON uses JavaScript syntax, but the JSON format is text only.

Text can be read & used as data format by any programming language.

The JSON format was originally specified by Douglas Crockford

Why use JSON?

Since the JSON format is text only, it can easily be sent to and from a server, and used as a data format by any programming language.

JavaScrxipt has a built in funciton to convert a string, written in JSON format, into native JavaScript objects: JSON.parse()

So, if you receive data from a server, in JSON format, you can use it like any other JavaScript object.
