Object Syntax

Imagine we have this object in JavaScript:

myObj = { "name" : "John", "age" : 30, "car" : null }

JSON objects are surrounded by curly braces { }

JSON objects are written in key/value pairs

Keys must be strings, and values must be valid JSON data type (string, number, object, array, boolean or null);

Keys and value are separated by a colon

Each key/value pair is separated by a comma

Accessing Object Values

x = myObj.name; // John

x = myObj["name"]; // John

Looping an Object

You can loop through object properties by using the for-in loop:

for (x in myObj) {
  document.getElementById("demo").innerHTML += x;
}

// name age car

In a for-in loop, use the bracket notation to access the property values:

for (x in myObj) {
  document.getElementById("demo").innerHTML += myObj[x];
}

// John 30 null

Nested JSON Objects

Values in a JSON object can be antoher JSON object

Nested Object

myObj = {
  "name" : "John",
  "age" : 20,
  "cars" : {
    "car1" : "Ford",
    "car2" : "BMW",
    "car3" : "Fiat",
   }
}

Access nested object

x = myObj.cars.car2;
x = myObj.cars["car2"];
x = myObj["cars"]["car2"];

Modify nested object

myObj.cars.car2 = "Mercedes";
myObj.cars["car2"] = "Mercedes";
myObj["cars"]["car2"] = "Mercedes";

Delete Object Property

Use the delete keyword to delete properties from a JSON object:

delete myObj.cars.car2;
delete myObj.cars["car2"];
delete myObj["cars"]["car2"];
