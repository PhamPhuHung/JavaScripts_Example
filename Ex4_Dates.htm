Date Objects

new Date()

Create new date object with the current date and time

var d = new Date(); // Mon Mar 15 2021 14:33:18 GMT+0700 (Indochina Time)

new Date(year, month, day, hour, minute, second, millisecond)

Create new date object with specific date and time

var d = new Date(2018, 11, 24, 10, 33, 30, 10); // Mon Dec 24 2018 10:33:30 GMT+0700 (Indochina Time)
var d = new Date(2018, 11, 24, 10, 33, 30); // Mon Dec 24 2018 10:33:00 GMT+0700 (Indochina Time)
var d = new Date(2018, 11, 24, 10, 33); // Mon Dec 24 2018 10:33:00 GMT+0700 (Indochina Time)
var d = new Date(2018, 11, 24, 10); // Mon Dec 24 2018 00:00:00 GMT+0700 (Indochina Time)
var d = new Date(2018, 11, 24); // Mon Dec 24 2018 10:00:00 GMT+0700 (Indochina Time)
var d = new Date(2018, 11); // Mon Dec 24 2018 10:33:30 GMT+0700 (Indochina Time)

new Date(datestring)

Create a new date object from datestring

Datestring format are described in the next chapter

var d = new Date("October 13, 2014 11:13:00"); // Mon Oct 13 2014 11:13:00 GMT+0700 (Indochina Time)

new Date(milliseconds)

Creata a new date object as zero time (1970-01-01) plus milliseconds

Note: 1 day (24 hours) is 86 400 0000 milliseconds

toString()

var d = new Date();
var str = d.toString(); // Mon Mar 15 2021 15:29:30 GMT+0700 (Indochina Time)

toUTCString()

var d = new Date();
var str = d.toUTCString(); // Mon Mar 15 2021 15:29:30 GMT

toDateString()

var d = new Date();
var str = d.toString(); // Mon Mar 15 2021

toString()

var d = new Date();
var str = d.toISOString(); // 2021-03-15T08:37:34.582Z

Date Format

ISO Dates

Complete Date: "2021-03-15" (YYYY-MM-DD)
Year and month: "2021-03" (YYYY-MM)
Year: "2021" (YYYY) Year, month, day, hour, minute, second: "2021-03-15T12:00:00Z" (YYYY-MM-DDTHH:MM:SSZ) (Z if time is UTC time)

Long Dates

"MMM DD YYYY"
("Mar 15 2021")
("March 15 2021")
("MARCH 15 2021")
("MArch, 15, 2021")

Date.parse(string)

Convert a Date to milliseconds

var msec = Date.parse("March 21, 2012"); // 1332262800000

Get Date Methods

Method Description
getFullYear() Get the year as four digit number (yyyy)
getMonth() Get the month as a number (0-11)
getDate() Get the day as a number (1-31)
getHours() Get the hour as a number (0-23)
getMinutes() Get the minute as a number (0-59)
getSeconds() Get the second as a number (0-59)
getMilliseconds() Get the millisecond as a number (0-999)
getDay() Get the weekday as a number (0-6)
Date.now() Get the time. ECMAScript 5. Act the same as getTime()
getTime() Get the time (milliseconds since 1970-01-01)

Method Description
getUTCFullYear() Same as getFullYear(), but retunrs the UTC year
getUTCMonth() Same as getMonth(), but retunrs the UTC month
getUTCDate() Same as getDate(), but retunrs the UTC date
getUTCHours() Same as getHours(), but retunrs the UTC hour
getUTCMinutes() Same as getMinutes(), but retunrs the UTC minute
getUTCSeconds() Same as getSeconds(), but retunrs the UTC seconds
getUTCMilliseconds() Same as getMilliseconds(), but retunrs the UTC milliseconds
getUTCDay() Same as getDay(), but retunrs the UTC day

Set Date Methods

Method Description
setFullYear(year, month, day) Set the year (optionally month and day)
setMonth() Set the month (0-11)
SetDate() Set the day (1-31)
SetHours() Set the hour (0-23)
SetMinutes() Set the minute (0-59)
SetSeconds() Set the second (0-59)
SetMilliseconds() Set the millisecond (0-999)
SetTime() Set the time (milliseconds since 1970-01-01)
